## Get started in three steps

<Steps>

### 1) Get an API token

Generate a **Personal Access Token** in the SageXAI dashboard. Copy the token value — you'll use it in the `Authorization` header:

```
Authorization: Bearer YOUR_TOKEN_HERE
```

<Callout>
For local development the base URL is **http://sagexai.test/**. In production, use your hosted API base (e.g., **https://api.sagexai.com/**).
</Callout>

### 3) Interpret the response

A successful response looks like:

```json
{
  "flagged": true,
  "payload": [
    {
      "start": 74,
      "end": 76,
      "text": "ι",
      "detector_type": "LLM03",
      "labels": ["Adversarial Unicode Homoglyph"],
      "message_id": 0
    }
  ],
  "breakdown": [/* optional diagnostics if requested */],
  "dev_info": { /* timing + rule metadata if requested */ }
}
```

- `flagged` — if **true**, at least one policy triggered.
- `payload` — list of matches with spans, labels, and which message they came from.
- `breakdown` / `dev_info` — turn on for troubleshooting during integration.
